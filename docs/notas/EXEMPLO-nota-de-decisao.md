# Nota de decisão — EXEMPLO

> Copie este modelo pra registrar decisões importantes do projeto.
> Nome do arquivo: `AAAA-MM-DD-assunto-curto.md` (ex.: `2026-06-13-escolha-do-banco.md`).

**Data:** 2026-06-13
**Status:** exemplo

## Contexto
Qual era a situação / o problema que exigiu uma decisão.

## Decisão
O que foi decidido, em uma frase clara.

## Por quê
As razões. Alternativas consideradas e por que foram descartadas.

## Consequências
O que muda daqui pra frente. O que fica mais fácil, o que fica mais difícil.
